📊 Diwali Sales Analysis

This project analyzes Diwali sales data to identify customer demographics, purchasing patterns, and high-performing product categories. Through data visualization and statistical analysis, valuable business insights are extracted to support data-driven marketing and sales strategies.

Key Features
Data cleaning and preprocessing
Handling missing values and data type conversion
Exploratory Data Analysis (EDA)
Customer segmentation by:
Gender
Age Group
State
Marital Status
Occupation
Product Category
Sales trend visualization using Matplotlib and Seaborn
Business insights and recommendations.
